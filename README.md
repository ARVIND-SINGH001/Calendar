# React Calendar UI Project (Assignment)

## Overview

This project is a responsive calendar UI built using **React** and **Tailwind CSS**. It focuses on usability, clean design, and interactive features commonly found in modern calendar applications.

## Features

### Calendar Grid

* Monthly calendar layout
* Proper weekday alignment
* Responsive grid structure
* Visual indicator (★) for dates with notes

**How this improves the project:**
Provides a clear and structured view of dates, making it easy for users to quickly identify important days and navigate the calendar efficiently.

### Notes System

* Click a date to add a note
* Modal popup for writing notes
* Notes stored per date
* Empty notes automatically removed
* Click outside to close without saving

**How this improves the project:**
Allows users to quickly attach reminders to specific days, improving usability and making the calendar more practical for daily planning.

### Responsive Layout

* Mobile-first design
* Dates and notes stack vertically on small screens
* Side-by-side layout on larger screens
* Equal width sections for visual balance

**How this improves the project:**
Ensures the calendar remains usable and visually balanced across different screen sizes, improving accessibility and user experience.

### Dynamic Image Header

* Background image changes dynamically
* Smooth transition effects
* Clean hero section styling

**How this improves the project:**
Improves visual appeal and keeps the interface engaging instead of static, creating a more modern application feel.

### Location Info Popup

* Info button on image header
* Popup displaying information about the place
* Click outside to close

**How this improves the project:**
Provides contextual information without cluttering the main UI, improving information accessibility while keeping the interface clean.

### UI Design Improvements

* Emerald color theme
* Modern modal styling
* Hover transitions
* Clean spacing and typography

**How this improves the project:**
Improves readability, visual hierarchy, and interaction feedback, making the application feel more polished and intuitive.

## Tech Stack

* React (Hooks & Functional Components)
* Tailwind CSS
* JavaScript (ES6)
* Vite

## Project Setup

### 1. Clone Repository

```bash
git clone https://github.com/ARVIND-SINGH001/Calendar.git
cd Calendar
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start Development Server

```bash
npm run dev
```

Project runs on:

```
http://localhost:5173
```
